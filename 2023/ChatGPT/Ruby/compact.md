# rubyのcompact

Ruby の compact! は、Array クラスのインスタンスメソッドで、配列から nil 要素を取り除きます。
このメソッドには ! がついているので、配列を直接変更する破壊的な操作を行います。
nil 要素が取り除かれた場合はその配列を返し、nil 要素が元からなかった場合は nil を返します。

例：

```
arr = [1, 2, nil, 4, nil]
arr.compact!  # => [1, 2, 4]
p arr         # [1, 2, 4]

arr2 = [1, 2, 3]
arr2.compact! # => nil
p arr2        # [1, 2, 3]
```

compact (末尾に ! がつかないバージョン) もありますが、これは元の配列を変更せずに新しい配列を返します。
